# taaaakeeen.github.io

https://taaaakeeen.github.io

Vite MPA (Multi-Page Application) で構築した個人サイト。

## Pages

- `/` — トップページ
- `/weather/` — 天気予報（気象庁API）
- `/note/` — 日記・メモ（Markdown）

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

GitHub Actions により `main` ブランチへの push 時に自動デプロイ。
